# Kaze Responsive UX

Prioritize mobile clarity because many visitors will review services, photos, and quote options on phones.

## Mobile Checklist

- Keep the first viewport instantly understandable: who Kaze is, what it offers, and how to quote.
- Use touch targets of at least 44px high.
- Avoid small overlay text on photos; enlarge, stack, or move below the image.
- Use stable dimensions for cards, galleries, icon modules, and CTAs.
- Keep hero text from covering essential image details.
- Test both `/` and `/technical` after shared CSS changes.

## Navigation

- Preserve `scroll-behavior: smooth`.
- Use predictable anchors and compact labels.
- Keep quote CTAs reachable from top navigation and lower sections.
- On mobile, avoid hidden critical actions behind decorative UI.

## Accessibility

- Use semantic headings in order.
- Keep contrast strong on image overlays.
- Provide useful alt text for meaningful images.
- Respect reduced motion for decorative animation.
