# INFO.md - Project Overview & General Information

## Project Name
KAZE Custom Apparel & Signs / Kaze Designs

## Business Description
KAZE is a local workshop in Watsonville, CA, specializing in lighted signs, custom uniforms, embroidery, screen printing, and business branding. They have over 10 years of experience serving local businesses, prioritizing human interaction, direct relationships ("customers are partners"), and premium finishes.

## Technology Stack
- **Version 1 / MVP (Current Production)**: Vanilla HTML5, CSS3, JavaScript. Integrates a custom Express.js static backend server.
- **Kaze Studio**: React 18, Vite 5, Tailwind CSS, TypeScript. Interactivity features canvas scaling, image importing, transformation, step-by-step layout flow, and image downloading. Built and compiled into the `/studio/` folder of the main site.
- **Version 2 (Future Roadmap)**: Next.js 14/15, Tailwind CSS, Framer Motion (currently preserved in `kaze-web/`).
- **Deployment**: Railway. Live production URL: [https://kazedesignswtv1-production.up.railway.app](https://kazedesignswtv1-production.up.railway.app)
- **Email System**: Custom Express backend endpoints using Resend API to deliver automated quotes.

## Core Directives & Aesthetics
1. **Premium Aesthetic**: Implement Glassmorphism, modern typography (Sora/Inter/Outfit), dark mode defaults, vibrant gold and crimson accents, and micro-animations.
2. **Localization & SEO**: Reinforce local presence in Watsonville, CA (Monterey Bay / Santa Cruz County). Maintain bilingual accessibility (Spanish/English).
3. **Mockup Precision**: Use high-quality base apparel generated by AI with proper drag-and-drop transformation tools for users.
