# INFO.md - Project Overview & General Information

## Project Name
KAZE Custom Apparel & Signs / Kaze Designs

## Business Description
KAZE is a local workshop in Watsonville, CA, specializing in lighted signs, custom uniforms, embroidery, screen printing, and business branding. They have over 10 years of experience serving local businesses, prioritizing human interaction, direct relationships ("customers are partners"), and premium finishes.

## Technology Stack
- **Version 1 / MVP (Current Production)**: Vanilla HTML5, CSS3, JavaScript. Uses CSS variables for theming, localized i18n logic via localStorage, and responsive grid layouts.
- **Version 2 (Next.js)**: React, Next.js 16 (App Router), Tailwind CSS, Framer Motion. 
- **Deployment**: Vercel (Production URL: https://kaze-site-local.vercel.app).
- **Other tools**: Remotion (for programmatic video presentations).

## Core Directives
1. **Premium Aesthetic**: Implement Glassmorphism, modern typography (Sora/Inter), vibrant accents (Gold/Crimson/Cyan), and subtle micro-animations.
2. **Localization & GEO/SEO**: Reinforce local presence in Watsonville, CA (Monterey Bay / Santa Cruz County). Keep language accessible, authentic, and bilingual (English/Spanish).
3. **No Placeholders**: Ensure all visual assets reflect the actual work or high-quality AI-generated mockups for signage and apparel.
